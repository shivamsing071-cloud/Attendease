# **App Name**: AttendEase

## Core Features:

- Configurable Timetable Grid: Create and configure a weekly timetable grid with customizable days and time slots.
- Slot Assignment: Assign subject name, type (Lecture/Lab), and color tag to each slot in the timetable.
- Slot Merging: Merge adjacent slots to represent labs or extended sessions. Store merged group ID in merged slots.
- Weekly Attendance Tracking: Toggle attendance status (Present/Absent) for each slot on a weekly basis, with bulk actions for marking all present/absent or clearing the week.
- Attendance Analytics: Calculate attendance percentage, missable classes, and recovery requirements for each subject based on attendance data. 
- Dashboard Visualization: Display subject-wise attendance analytics using cards, showing subject name, attendance percentage, attended/total slots, and safe/danger indicators.
- Text prediction alerts: Leverage AI to intelligently make predictions based on existing text inputs within the slots of the current and future schedule, turning it into a valuable tool to optimize schedule design. It helps in creating coherent schedule planning across the calendar and across the courses.

## Style Guidelines:

- Primary color: Teal (#008080) to represent focus and productivity.
- Background color: Light teal (#E0F8F7), a desaturated version of the primary for a calm, focused backdrop.
- Accent color: Yellow-green (#B0E293) to provide contrast and signal key interactive elements.
- Body font: 'PT Sans', a modern and friendly sans-serif for clear and accessible text.
- Headline font: 'Space Grotesk', a strong and versatile sans-serif for headers and titles, giving a techy edge.
- Use clear, minimalist icons from Material Design to represent subject types, attendance status, and actions.
- Implement a scrollable timetable grid with fixed time columns for easy navigation.
- Use subtle animations for merging slots and toggling attendance status to provide visual feedback and improve user experience.
